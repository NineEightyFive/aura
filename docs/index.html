About AURA

This app was made in consultation with Landmark Collegeâ€™s radio station, WLMC Radio: The Finn. The station was looking for a more streamlined and effective way to organize, tag, and convert files from vox pops, recording sessions, and live shows.
Aura is a Java-based program that uses Java Swing to implement a GUI interface and SwingWorker threads to handle file conversion without locking up the program. conversion and metadata modding is achieved using the JAVE2 AND JAudioTagger Libraries. The converter engine has error handling, duplicate checking, and a nondestructive baseline.

Features

  • Metadata

    Edit any field of metadata of an audio file, including title, artist, album, etc.

  • Wifi-Free

    No internet connection required. Convert files from anywhere, in the field, and on the go!

  • No Libraries Required

    The program download comes with all required prerequisites bundled, no hassle with additional downloads.

Ready to Use AURA?

Available on Windows, Mac (ARM64, x64), and Linux, be ready to conquer your audio conversion and metadata workflow with AURA.
